# 「포켓 에그: 나만의 반려몬」 디자인 브레인스토밍

## 세 가지 스타일 방향

### 1. Cozy Nursery (코지 너서리)
부드러운 파스텔 톤과 둥근 형태를 활용한 아기방 콘셉트. 따뜻하고 안전한 느낌을 주어 반려몬을 돌보는 양육 감정을 극대화한다.
- **확률:** 0.04

### 2. Pixel Retro (픽셀 레트로)
90년대 다마고치 원본의 도트 그래픽을 현대적으로 재해석한 레트로 스타일. 노스탤지어를 자극하면서도 선명한 색감으로 신선함을 준다.
- **확률:** 0.03

### 3. Kawaii Pop (카와이 팝)
일본 카와이 문화와 한국 캐릭터 디자인을 결합한 팝 아트 스타일. 선명한 원색 포인트와 귀여운 이모티콘 기반 표현으로 10~20대 타겟에 어필한다.
- **확률:** 0.07

## 선택된 방향: Cozy Nursery (코지 너서리)

### Design Movement
**Soft Morphism + Illustrated Warmth** — 뉴모피즘의 부드러운 입체감과 수채화풍 일러스트를 결합한 양육 게임 특화 디자인. 터치할 때마다 포근함이 느껴지는 촉각적 UI를 지향한다.

### Core Principles
1. **Rounded Softness** — 모든 UI 요소는 완전한 라운드 또는 pill 형태를 기본으로 한다. 날카로운 모서리는 존재하지 않는다.
2. **Layered Depth** — 카드와 버튼에 부드러운 그림자와 내부 하이라이트를 적용하여 마치 쿠션처럼 눌리는 느낌을 준다.
3. **Living Palette** — 반려몬의 감정 상태에 따라 배경 색조가 미세하게 변화하여 공간 전체가 살아있는 느낌을 준다.
4. **Touch Intimacy** — 모든 인터랙션은 직접 만지는 듯한 촉각적 피드백을 제공한다.

### Color Philosophy
따뜻한 크림색 배경 위에 연한 피치, 민트, 라벤더가 어우러진 파스텔 팔레트를 사용한다. 이는 아기방의 안전하고 포근한 분위기를 연출하며, 사용자가 오래 머물러도 눈이 피로하지 않도록 설계되었다.

| 용도 | 색상 | OKLCH |
|---|---|---|
| 배경 (크림) | #FFF8F0 | oklch(0.98 0.015 80) |
| 주요 액센트 (피치) | #FFB088 | oklch(0.80 0.12 50) |
| 보조 액센트 (민트) | #A8E6CF | oklch(0.87 0.10 160) |
| 포인트 (라벤더) | #C8B6FF | oklch(0.80 0.12 290) |
| 텍스트 (다크 브라운) | #4A3728 | oklch(0.35 0.05 60) |
| 서브 텍스트 | #8B7355 | oklch(0.55 0.06 70) |

### Layout Paradigm
**Vertical Card Stack** — 모바일 세로 화면에 최적화된 수직 카드 스택 구조. 반려몬이 화면 중앙에 크게 위치하고, 상태 정보는 상단에 떠있는 풍선처럼, 액션 버튼은 하단에 부드러운 도크 형태로 배치된다. 배경은 방 전체를 보여주는 일러스트로 채워진다.

### Signature Elements
1. **Bubble Gauge** — 상태 게이지가 비눗방울 형태로 떠다니며, 수치에 따라 크기와 색상이 변한다.
2. **Cushion Button** — 버튼을 누르면 쿠션처럼 눌리는 3D 효과와 함께 부드러운 바운스 애니메이션이 재생된다.
3. **Paw Print Trail** — 화면 전환 시 발자국이 이동 경로를 따라 나타나는 트랜지션 효과.

### Interaction Philosophy
모든 터치는 "부드럽게 누르는" 촉각적 경험을 제공한다. 버튼은 누를 때 살짝 가라앉고(scale 0.95), 놓을 때 탄성 있게 복원된다. 반려몬을 터치하면 물결처럼 반응이 퍼지며, 드래그 동작은 부드러운 관성을 가진다.

### Animation
- **진입 애니메이션:** 요소들이 아래에서 위로 부드럽게 떠오르며 나타남 (200-300ms, ease-out)
- **상태 변화:** 게이지 변화 시 물이 차오르는 듯한 유체 애니메이션
- **반려몬 반응:** idle 상태에서 미세한 호흡 애니메이션, 터치 시 기쁨 표현
- **화면 전환:** 페이드 + 슬라이드 조합, 발자국 트레일 이펙트
- **보상 연출:** 별과 하트가 터지는 파티클 이펙트

### Typography System
- **Display / 타이틀:** "Galmuri11" (한글 픽셀 감성 폰트) — 게임 제목과 주요 헤딩에 사용
- **Body / 본문:** "Pretendard" — 깔끔하고 가독성 높은 한글 본문용
- **숫자 / 수치:** "Nunito" — 둥글고 친근한 느낌의 숫자 표현

### Brand Essence
> 매일 돌아오고 싶은 나만의 포근한 알 세계 — 바쁜 일상 속 잠깐의 힐링을 원하는 모바일 유저를 위한, 감정이 살아있는 가상 반려 육성 게임.

**성격 형용사:** 포근한(Cozy), 생동감 있는(Lively), 애착이 가는(Endearing)

### Brand Voice
헤드라인과 CTA는 반려몬이 직접 말하는 듯한 1인칭 톤을 사용한다. 짧고 귀엽지만 유치하지 않은 표현을 지향한다.
- 예시 1: "배고파... 오늘은 뭐 먹을까?"
- 예시 2: "같이 놀자! 공 던지기 어때?"

### Wordmark & Logo
알 모양 실루엣 안에 잠든 반려몬의 귀가 살짝 보이는 심볼. 알의 윤곽선은 부드러운 크림색 그라데이션으로 처리하며, 상단에 작은 균열이 있어 곧 부화할 것 같은 기대감을 준다.

### Signature Brand Color
**Pocket Peach (#FFB088)** — 따뜻한 복숭아색. 크림 배경 위에서 눈에 띄면서도 부드러운 인상을 주는 이 브랜드만의 시그니처 컬러.
